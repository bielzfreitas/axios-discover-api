Foi usada a API node-api-discover-main feita pela Jakeliny (https://github.com/jakeliny)
-> Buscar no Temrinal
-> Fazer o "npm i"
-> Após, digitar "npm start"
-> A API vai rodar na porta "5500"
-> No navegador, o endereço é: "localhost:5500/api"
